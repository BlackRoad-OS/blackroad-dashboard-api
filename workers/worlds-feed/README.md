## 1. Overview

The Worlds Feed worker provides RSS 2.0 and Atom 1.0 syndication feeds for AI world artifacts generated by the BlackRoad Pi fleet. Subscribe to receive live updates as new worlds, lore entries, and code artifacts are created.

---

## 2. API Endpoints

| Method | Path | Content-Type | Description |
|---|---|---|---|
| `GET` | `/feed.rss` | `application/rss+xml` | RSS 2.0 feed (latest 50 artifacts) |
| `GET` | `/rss` | `application/rss+xml` | RSS 2.0 feed (alias) |
| `GET` | `/feed.atom` | `application/atom+xml` | Atom 1.0 feed (latest 50 artifacts) |
| `GET` | `/atom` | `application/atom+xml` | Atom 1.0 feed (alias) |
| `GET` | `/health` | `application/json` | Service health check |

## 3. Feed Formats

### RSS 2.0

```xml
<?xml version="1.0" encoding="UTF-8"?>
<rss version="2.0">
  <channel>
    <title>BlackRoad Worlds Feed</title>
    <link>https://worlds.blackroad.io</link>
    <description>AI-generated world artifacts from the BlackRoad fleet</description>
    <item>
      <title>Crystal Depths [world@aria64]</title>
      <link>https://github.com/BlackRoad-OS-Inc/blackroad-agents/blob/main/worlds/...</link>
      <guid>20260301_120000_world_crystal-depths</guid>
      <pubDate>Sat, 01 Mar 2026 12:00:00 +0000</pubDate>
    </item>
    <!-- ... -->
  </channel>
</rss>
```

### Atom 1.0

```xml
<?xml version="1.0" encoding="UTF-8"?>
<feed xmlns="http://www.w3.org/2005/Atom">
  <title>BlackRoad Worlds</title>
  <id>https://worlds.blackroad.io/</id>
  <updated>2026-03-01T12:00:00Z</updated>
  <entry>
    <id>urn:blackroad:world:20260301_120000_world_crystal-depths</id>
    <title>Crystal Depths</title>
    <link href="https://github.com/..."/>
    <updated>2026-03-01T12:00:00Z</updated>
    <author><name>aria64</name></author>
  </entry>
  <!-- ... -->
</feed>
```
